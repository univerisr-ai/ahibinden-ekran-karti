## 🛠️ Kurulum Adımları

### 1. Dosyaları Yükle
Bu `v1_nodriver_cffi` klasörünü sunucunuza yükleyin.

### 2. Sistem Paketlerini Kur
```bash
sudo apt-get update
sudo apt-get install -y xvfb python3-pip python3-venv
```

### 3. Cloudflare WARP Kur (ÜCRETSİZ IP MASKELEMESİ)
```bash
sudo bash warp_setup.sh
```
Bu script otomatik olarak:
- Cloudflare WARP'ı kurar
- SOCKS5 proxy modunda yapılandırır (port: 40000)
- Bağlantıyı test eder
- Otomatik başlatmayı etkinleştirir

Kurulumdan sonra test:
```bash
# WARP durumunu kontrol et
warp-cli status

# WARP IP'sini gör
curl --socks5 127.0.0.1:40000 https://ifconfig.me

# Normal IP'yi gör (farklı olmalı)
curl https://ifconfig.me
```

### 4. Python Bağımlılıklarını Kur
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 5. .env Dosyasını Ayarla
`.env` dosyasını açıp Telegram Token'ınızı yapıştırın.

### 6. Botu Başlat
```bash
python bot_controller.py
```

### 7. Otomatik Cookie Yenileme (Cron Job)
Cookie'lerin her 45 dakikada otomatik yenilenmesi için:
```bash
crontab -e
```
Şu satırı ekleyin:
```
*/45 * * * * cd /opt/ai-gez && source .venv/bin/activate && python cookie_generator.py >> /var/log/cookie_gen.log 2>&1
```
